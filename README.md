# jquery-sample
jQueryカスタムプラグインのサンプルです。  
スライドショーとして動作します。

## Usage

[デモページ](http://code.rlated.net/jquery/plugin/ "jQueryプラグインのサンプル")を参考にしてください。

_デフォルト_

    $('#target').sample();

_オプションを指定する場合_

    $('#target').sample({duration: 1000});

_オプションの説明_

- duration: 表示する時間 (デフォルト値は1000)
